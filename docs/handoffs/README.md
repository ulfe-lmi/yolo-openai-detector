# Handoffs

Use this directory for strategic handoffs when context becomes long or the project moves to a new chat/session.

Suggested template:

```markdown
# Handoff — YYYY-MM-DD

## Current truth
- Main branch:
- Open PRs:
- CI status:

## Product goal
- ...

## Implemented
- ...

## Missing
- ...

## Non-negotiable rules
- CPU-only
- Single image only
- No tracking/video/segmentation/background jobs
- OpenAI-compatible `/v1/models` and `/v1/chat/completions`

## Next recommended task
- ...

## Do not do next
- ...
```

A handoff is a memory aid, not authoritative over the live repository.
